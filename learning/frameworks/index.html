JS 库

工具库

  • jQuery 流行的 JS 工具库，提供了 DOM 元素查找，事件绑定，Ajax 等 source
  • Zepto 类 jQuery，使用方法基本一致，针对移动端优化，库体积很小 source
  • Underscore 流行的函数库，提供了集合，数组，对象，字符串，函数等的常用函数 source
  • Lodash 类 Underscore 的函数库，修正了 underscore 的一些问题，亦有一些优化 source
  • Momentjs 专注于时间／日期操作的库，如格式转换，值校验，日期计算，全球时间格式支持等 source
  • iScroll 专注于元素滚动的库，可使元素平滑滚动。使用该库在移动设备上可实现下拉刷新效果source

MVC，MVVM，etc..

  • Vue 国人开发的 MVVM 库，轻量易学最近愈加流行 source
  • React Facebook 开发的 MVVM 库，相比 Vue 上手难度略高source
  • Angularjs Google 开发的 MVVM 库 source
  • Spine 使用 coffeescript 编写的 MVC 库 source
  • Backbones 老牌的 MVC 库 source

模块化管理工具

  • requirejs国际流行的 AMD 风格的模块化管理工具 source

CSS 框架

  • Bootstrap Twitter 出品，组件丰富而复杂，使用广泛。source
  • weui 微信官方提供的 UI 库 source
  • purecss 雅虎出品的 css 库，简约，包含常用组件，响应式设计，易于集成到现有项目。source
  • Primer Github 出品，从 github 官网提取出的 css 库。 source
  • Semantic UI 自带 UI 组件比 twitter 丰富，但其 css 系统略复杂。source
  • Kube

组件库

  • artdialog 国人开发的对话框组件source
  • datetimepicker 日期时间选择插件，依赖 bootstrap source

其他资源素材

图标

  • fontello 字体图标生成工具。第三方图标提供了很丰富的字体，但是往往几百个图标中只用到了其中少部分图标，使用该工具可自定义生成只包含自己使用图标的字体文件，大大减少字体图标体积。该工具支持从众多免费开源的字体图标中选择，包括 Font awesome，Entypo, Typicons, Iconic 等等
  • Font awesome 拥有超过 600 个免费开源的字体图标，囊括了绝大部分常用图标
  • Line awesome 和 Font awesome 类似的字体图标库，图标风格略有差异。由 icons 8 提供
  • icons 8 包含四万多多免费图标。免费使用的图标有尺寸限制，且需要在网站添加其链接。

自动化测试

  • mocha 流行的自动化测试框架 source
  • chai 流行的断言库, 可供测试框架调用 source
  • phantomjs 在 nodejs 中模拟浏览器环境, 可用于做浏览器端端自动化测试 source
